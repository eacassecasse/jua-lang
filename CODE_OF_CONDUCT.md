# Code of Conduct

## Our Commitment

We are committed to maintaining a constructive and respectful environment for all contributors, regardless of experience level.

Jua is designed to be both:

* an educational language
* a production-grade system

This requires collaboration across different skill levels.

---

## Expected Behavior

Participants are expected to:

* be respectful in discussions
* provide constructive feedback
* focus on technical arguments
* avoid personal criticism
* be patient with learners

---

## Unacceptable Behavior

Includes:

* harassment of any kind
* dismissive or hostile responses
* gatekeeping technical knowledge
* disrespectful communication
* intentional disruption of discussions

---

## Enforcement

Maintainers may:

* request behavior correction
* remove comments or contributions
* temporarily restrict participation
* escalate to GitHub moderation tools if necessary

---

## Scope

This Code of Conduct applies to:

* issues
* pull requests
* discussions
* reviews
* any project communication channels
